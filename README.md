# HelpSIN - Projeto de estudo no NotebookLM
Este projeto utiliza o **NotebookLM** como ferramenta de **Inteligência Artificial Generativa** aplicada para consultar regras e normas, referentes a sinalização horizontal e vertical, nos manuais do **CONTRAN** (Conselho Nacional de Trânsito). O objetivo é **acelerar** a execução dos projetos de sinalização viária, reduzindo o tempo que seria gasto procurando informações em diferentes manuais.

O NotebookLM desenvolvido neste projeto pode ser acessado no link abaixo:

https://notebooklm.google.com/notebook/2ca28cf0-1f66-40bb-8e66-2081c6b75730

---

## Fontes Utilizadas
As respostas do NotebookLM são geradas **exclusivamente** a partir das fontes carregadas, evitando alucinações nas respostas.
- Volume I - Manual de Sinalização Vertical de Regulamentação-2022.pd;
- Volume II - Manual de Sinalizacao Vertical de Advertencia-2022.pdf;
- Volume III - Manual de Sinalizacao Vertical Indicativa-2022.pdf;
- Volume IV - Manual de Sinalizacao Horizontal-2022.pdf;
- Volume V - Manual de Sinalização Semaforica-2022.pdf;
- Volume VI - Manual de Dispositivos Auxiliares-2022.pdf;
- Volume VIII - Manual de Sinalização- Cicloviário-2022.pdf.
